# StegoChat 🛡️🔐  
A modern, privacy-focused chat application with built-in image steganography.

StegoChat allows users to hide secret messages inside images, send encrypted chats, create group chats, and manage media — all inside a clean and modern UI.  
This project is built as a secure, end-to-end controlled communication tool.

---

## 🚀 Features

### 🔒 Steganography
- Hide secret text messages inside images  
- Extract hidden messages from encoded images  
- Lossless encoding/decoding algorithm  
- Secure message handling and clean UI display  

### 💬 Real-Time Chat System
- Start **New Chat**  
- Supports **Group Chat**
- Media menu: Open, Download, Info, Delete  
- Smooth message rendering & clean layout  

### 🏞️ Image Management
- Upload images  
- Add hidden text  
- Download encoded images  
- Detailed image info popup  
- Delete button implemented (working in New Chat + Group Chat)

### 🎨 Modern UI
- Responsive layout  
- Sidebar navigation  
- Animated buttons & transitions  
- Logo: **StegoChat (modern & tech-focused)**  
- Dark mode friendly  

### 🛠️ Technology Stack
- **HTML5, CSS3, JavaScript**  
- Steganography script (custom)  
- DOM-based front-end logic  
- Local storage handling  
- Modular and maintainable project structure

## 🧪 How It Works

### 🟢 Hide Message
1. Upload an image  
2. Enter secret text  
3. Click **Encode**  
4. Download/export the Stego Image  

### 🔵 Reveal Message
1. Upload encoded image  
2. Click **Decode**  
3. Hidden text appears instantly
